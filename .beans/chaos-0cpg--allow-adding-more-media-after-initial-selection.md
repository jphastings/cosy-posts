Currently you can only select media once. Add the ability to add more media items after the initial selection.

The 'Tap to select photos or videos' placeholder should have a dotted border to make it look like a drop target / action area. After media is selected, the same dotted-border box should appear as the last item in the swipeable media preview, allowing the user to tap it to add more items via PHPicker. New items are appended to the existing selection.
